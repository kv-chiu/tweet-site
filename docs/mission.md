## 定位

社交媒体是信息流，我们是图片展。

把散落在 Twitter、Instagram 等社交媒体上的高质量图片整理成独立的图片站，让粉丝像刷相册一样沉浸式浏览，一键分享给同好。

目标受众：追星粉丝、coser 粉丝、喜欢收藏社交媒体图片的人。

## User Story 1: 粉丝想把收藏的图片做成图片站

> 作为一个追星/追 coser 的粉丝，
> 我想把散落在社交媒体上的高质量图片整理成一个独立的图片站，
> 以便像刷相册一样沉浸式浏览，并且能一键分享给同好。

- 记录内容 ID（JSON 格式），存储在 Git 仓库的 data 分支中。
- 支持多数据源：Twitter 推文、Instagram 帖子。
- 一键部署到 Vercel，方便地分享给朋友看。
- 瀑布流布局沉浸浏览完整内容。
