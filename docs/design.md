## 定位

推特是信息流，我们是图片展。

把散落在推特上的高质量图片整理成独立的图片站，让粉丝像刷相册一样沉浸式浏览，一键分享给同好。

目标受众：追星粉丝、coser 粉丝、喜欢收藏推特图片的人。

## User Story 1: 粉丝想把收藏的推文图片做成图片站

> 作为一个追星/追 coser 的粉丝，
> 我想把散落在推特上的高质量图片整理成一个独立的图片站，
> 以便像刷相册一样沉浸式浏览，并且能一键分享给同好。

- 记录推文 ID（JSON 格式），存储在 Git 仓库中。
- 一键部署到 Vercel，方便地分享给朋友看。
- 瀑布流布局沉浸浏览完整推文，相册布局专注看图。
- Lightbox 大图浏览，悬浮查看原始推文。
